# MusicCalculator
Music Calculator is an audio plugin for Mac OSX created using the [JUCE framework](https://github.com/julianstorer/JUCE).
## Features
* Convert fraction of tempo to hz/ms
* Convert note to hz
* Sync tempo to DAW
* Sync note to MIDI
* Copy conversion to clipboard
* Resizable GUI
